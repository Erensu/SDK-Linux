# indemind-sdk-Linux
Binocular Vision Inertial Module


INDEMIND双目视觉惯性模组采用全局快门的2X1280X800@50FPS高清摄像头，可提供水平120°、垂向75°视场角，结合高帧率6轴IMU传感器，为SLAM等算法提供强有力前端数据采集能力。


INDEMIND双目视觉惯性模组SDK提供如下接口：
1）设备信息；
2）图像数据；
3）IMU数据；
4）SLAM；
5）深度图。
详细信息参考《双目惯性模组用户使用说明》。


支持平台：支持Linux18.04平台。


模组数据采集软件，采集模组图像及IMU数据，并保存成EuRoC数据集格式，方便用户使用。
详细信息参考《模组数据采集软件说明文档》
支持Linux18.04平台。
